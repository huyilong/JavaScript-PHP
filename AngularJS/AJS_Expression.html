AngularJS extends HTML attributes with Directives, and binds data to HTML with Expressions. AngularJS is distributed as a JavaScript file, and can be added to a web page with a script tag: AJS extends HTML by adding "directives" in tags, adding "expression" to bind data to HTML, adding "controller" to control the application AJS - a package for JS like bootstrap for CSS *** The ng-app directive tells AngularJS that the
element is the "owner" of an AngularJS application. AngularJS Expressions AngularJS expressions are written inside double braces: {{ expression }}. AngularJS expressions bind data to HTML the same way as the ng-bind directive. AngularJS will "output" data exactly where the expression is written. AJS Variable using ng-init="quantity=1;cost=5"

Total in dollar: {{ quantity * cost }}

AngularJS Expressions AngularJS expressions are written inside double braces: {{ expression }}. AngularJS expressions binds data to HTML the same way as the ng-bind directive. AngularJS will "output" data exactly where the expression is written. AngularJS expressions are much like JavaScript expressions: They can contain literals, operators, and variables. Example {{ 5 + 5 }} or {{ firstName + " " + lastName }} If you remove the ng-app directive, HTML will display the expression as it is, without solving it.
