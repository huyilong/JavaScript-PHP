XLink Attribute Reference Attribute Value Description 1.xlink:actuate onLoad onRequest other none Defines when the linked resource is read and shown: onLoad - the resource should be loaded and shown when the document loads onRequest - the resource is not read or shown before the link is clicked 2.xlink:href URL Specifies the URL to link to 3.xlink:show embed new replace other none Specifies where to open the link. Default is "replace" 4.xlink:type simple extended locator arc resource title none Specifies the type of link
