XML Errors Will Stop You Errors in XML documents will stop your XML applications. The W3C XML specification states that a program should stop processing an XML document if it finds an error. The reason is that XML software should be small, fast, and compatible. HTML browsers will display HTML documents with errors (like missing end tags).
