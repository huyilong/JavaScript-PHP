XSL stands for EXtensible Stylesheet Language, and is a style sheet language for XML documents. (XML is just describe data XSL is EXtensible Stylesheet Language to define style sheet for XML documents XSLT is using XPath to select different element nodes in XML and transfer is into XHTML ) XSLT stands for XSL Transformations. In this tutorial you will learn how to use XSLT to transform XML documents into other formats, like XHTML. (XHTML is similar to HTML, but it is much more strict compared with HTML like: it requires !DOCTYPE and attribute not be capitalized and must tag paired) XSLT Elements Description of all the XSLT elements from the W3C Recommendation, and information about browser support. XSLT Functions XSLT includes over 100 built-in functions. There are functions for string values, numeric values, date and time comparison, node and QName manipulation, sequence manipulation, Boolean values, and more. XQuery 1.0, XPath 2.0, and XSLT 2.0 share the same functions library.
